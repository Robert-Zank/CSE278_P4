An Order Query Application

QueryPage

This website is an Order Query Application that interacts with a MySQL database. The first quary allows the user to enter a shipping status. It will then return the order date, required by date, and the customerNumber for all orders with the entered shipping status. The second query allows users to select a customerNumber from a dropdown menu, and retrieve relevant order information. Upon submitting the query, the application fetches the data from the database and displays it in a table format, showing the status and the number of orders associated with the selected customerNumber.
